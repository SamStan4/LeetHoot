# LeetHoot

# TO RUN (needs 3 terminals)

### 1. Start the problem api server  
```bash
cd problem-api
npm install
npm run start
```

### 2. Start the main server  
```bash
cd server
npm install
npm run start
```

### 3. Start the client html renderer
```bash
cd client
npm install
npm run dev
```

### NOTE:  
No database work needed, app comes packaged with SQLite instance
