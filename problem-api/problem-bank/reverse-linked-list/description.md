# Reverse Linked List

Given the `head`of a singly linked list, reverse the list, and return *the reversed list*.

**Example 1:**

**Input:** arr = [ ]<br>
**Output:** []

**Example 2:**

**Input:** arr = [1,2,3,4,5]<br>
**Output:** [5,4,3,2,1]

**Example 3:**

**Input:** s = [1,2]<br>
**Output:** [2,1]

**Constraints:**

- The number of nodes in the listt is in the range`[0, 20000]`<br>
- `0<= Node.val <= 20000`
