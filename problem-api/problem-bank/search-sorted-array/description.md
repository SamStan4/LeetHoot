# Search Sorted Array
  
Given a sorted array `arr` of numbers and a `target`, determine if `target` is in the `arr`.  
  
**Example 1:**  
  
**Input:** arr = [ ], target = 5  
**Output:** false  
  
**Example 2:**  
  
**Input:** arr = [1,2,3,4,5], target = 4  
**Output:** true  
  
**Example 3:**  
  
**Input:** s = [1,2,3,4,5], target = 6  
**Output:** false  
  
**Constraints:**  
  
- `0 <= arr.length <= 10^4`
- `0 <= target <= 10^4`
- `target` appears at most once in the array.
